# Dynamic creation of elements

**Assignment**

+ Create an application with a simple form containing only one &lt;input&gt; element and one &lt;button&gt; element.
+ Write code so that filling in the input and clicking the button will append a new &lt;li&gt; element, containing the text provided in the input, to a created earlier &lt;ul&gt; element.
+ Additionally, you can make sure that empty inputs will not be processed.
